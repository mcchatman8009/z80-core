# Z80 Core

## Overview
A simple C++ Z80 Cpu library that that can be
used to emulate various systems that use the
original Z80 Cpu. For example the Sega Master System
& and the Sega Game Gear.

## Getting Started

### Build Requirements
* CMake 3+
* C++17

### Steps to build the Project
```
mkdir build
cd ./build
cmake  ..
make all
```


### Testing
```
mkdir build
cd ./build
cmake  ..
make test
```
